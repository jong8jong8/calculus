```
# EXAMPLE 5.4 (Geogebra's List starts from 1 not 0)

f(x)=x^(2)
n=4
a=0
b=2
Δx=((b-a)/(n))
xi=Sequence(a+i Δx,i,0,n)
partition=(xi,0)
fvalue=Sequence(f(xi(i)),i,1,n+1)
fpoints=(xi,fvalue)
result=fvalue Δx
L=Sum(result(i),i,1,n)
R=Sum(result(i),i,2,n+1)
```
